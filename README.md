# Overview | 项目简介

WebInsight Agent 是一个实验性的 Autonomous AI System，旨在模拟多个 AI Agent 协同分析网站的推理流程。

WebInsight Agent is an experimental autonomous AI system designed to simulate collaborative reasoning between multiple specialized AI agents for website analysis.

该平台基于 GPT-4o，对网站进行：

The platform leverages GPT-4o to autonomously evaluate:

- SEO Structure（SEO 结构）
- UX Hierarchy（用户体验层级）
- Conversion Funnels（转化漏斗）
- Growth Opportunities（增长机会）
- Website Positioning（网站定位）

系统通过以下方式可视化 AI 推理过程：

The system visualizes AI reasoning workflows through:

- Agent Timelines
- Streaming Analysis Logs
- Multi-Agent Orchestration
- Executive Dashboard Reporting

该项目主要探索 Frontend Autonomous AI Workflow 的未来交互形态。

This project explores the future interaction paradigm of frontend autonomous AI systems.
